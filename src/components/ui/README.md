# components/ui/

shadcn/ui primitives, copied into the repo (not installed as an opaque
dependency) and themed exclusively through the CSS custom properties in
`src/app/globals.css` — never a hardcoded color, spacing, or radius value
(Phase 1 §7, Phase 4's design-system rules).

Populated in Stage 4 (Design System) and Stage 5 (Shared Components).
